\sechead{1. 背景と目的}
\begin{itemize}
  \item 鏡面反射物体は光沢・映り込みにより外観が不安定 → 教師なし手法に深刻な課題
  \item SAVi-DINOSAUR: 凍結ViT特徴量 + Slot Attention で教師なしセグメンテーション
  \item 3バックボーン (DINOv2 / DINOv1 / CLIP) と金属・非金属 2素材で比較評価
  \item Slot数 Kの感度とエントロピー・時系列一貫性で失敗モードを分析
\end{itemize}
\vspace{1mm}
\colorbox{lightyellow}{\parbox{\dimexpr\linewidth-2\fboxsep}{%
  \textbf{課題設定:} 凍結ViT特徴量の質がセグメンテーション性能を左右するか？
  鏡面反射・光沢面に対して各バックボーンはどのような崩壊モードを示すか？
}}\vspace{3mm}

%%% sec4
\sechead{4. DINOv2 高FG-ARIの要因}
\begin{center}
  \includegraphics[width=\linewidth]{figures/paper_3backbone_k11_comparison.png}
\end{center}
\figcap{図2: 3バックボーン $K{=}11$ 比較（金属シーン）。上: DINOv2、中: DINOv1、下: CLIP}
\vspace{1mm}
\colorbox{lightyellow}{\parbox{\dimexpr\linewidth-2\fboxsep}{%
  \textbf{Goldilocks 特性:} $K{=}9$ 付近で FG-ARI が最大化後に飽和する。
  スロット数が多すぎると物体が分割され、少なすぎると背景と混合する。
}}\vspace{2mm}
\begin{itemize}
  \item DINOv2: スロットが物体境界に整合、金属面・反射ハイライトも局在化成功
  \item DINOv1: チャンネル正規化後も背景との分離が不安定（図中段）
  \item CLIP: 言語的意味論が優先 → テクスチャ境界を無視しスロット崩壊（図下段）
\end{itemize}
\vspace{3mm}

\sechead{5. 定性評価: スロット可視化}
\begin{center}
  \includegraphics[width=\linewidth]{figures/paper_dinov2_k11_slots.png}
\end{center}
\figcap{図3: DINOv2 $K{=}11$ の複数シーン（入力・GT・予測）}
\begin{itemize}
  \item 代表4シーン（金属2・混合2）の FG-ARI 範囲: 49.7\%〜61.2\%
  \item DINOv2 は全体として境界分割に優れるが、鏡面ハイライト部分ではスロット混同が依然として発生する
  \item 鏡面反射像がスロット割り当てエントロピーを局所的に増大させる要因として観察された
\end{itemize}

\sechead{6. エントロピー分析（300シーン）}
\begin{center}
  \includegraphics[width=\linewidth]{figures/uncertainty_showcase.png}
\end{center}
\figcap{図4: スロット割当エントロピー（上段: High Entropy〔不確実〕、下段: Low Entropy〔確信〕）}
\begin{itemize}
  \item 金属面の映り込み領域でエントロピーが局所的に増大
  \item DINOv2 は高エントロピー領域でも正しい割当を維持
  \item DINOv1・CLIP は高エントロピー領域で周辺スロットへの漏れが顕著
  \item 物体数が少ないシーン（余剰スロット多）でエントロピー高（$p{<}10^{-4}$）
\end{itemize}

%%% sec8
\sechead{8. まとめと今後の課題}
\begin{itemize}
  \item DINOv2 が鏡面シーンで DINOv1・CLIP を一貫して上回ることを定量的に確認
  \item Goldilocks 特性: $K{=}9$ が最適、過少・過多でともに FG-ARI 低下
  \item エントロピー分析で余剰スロットが崩壊の定量的指標となることを確認
  \item 時系列追跡で SAVi-DINOSAUR の動画理解への適用可能性を示唆
\end{itemize}
\vspace{2mm}
\begin{itemize}
  \item 鏡面反射に特化した損失関数・データ拡張の設計が必要
  \item DINOv1・CLIP でのスロット崩壊の根本原因（特徴空間構造）の解明
  \item 実物体・実シーンへのゼロショット汎化性能の検証
\end{itemize}
\vspace{2mm}
\colorbox{lightyellow}{\parbox{\dimexpr\linewidth-2\fboxsep}{%
  \textbf{Limitations:} 本研究は MOVi-A の合成データのみを対象としており、
  実世界の鏡面物体への汎化は未検証。公式チェックポイントが存在せず
  完全な再現実験には独自学習が必要。
}}
