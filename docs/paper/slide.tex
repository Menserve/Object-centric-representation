\noindent
\sechead{4.\ まとめと工学的教訓}
\vspace{1mm}

\noindent{\scriptsize
\textbf{(1)}\,DINOv2は``Goldilocks特性''によりSlot Attentionと最適な相性 $\to$ K=11でFG-ARI\,\textbf{0.470}\hfill
\textbf{(2)}\,スロット数はバックボーン特性に強く依存（DINOv2のみK=11が有効）\\[0.5mm]
\textbf{(3)}\,工学的知見：2層MLP射影，ch-norm損失（DINOv1），勾配detach（CLIP），温度$\tau{=}0.5$\hfill
\textbf{(4)}\,16$\times$16パッチ解像度の壁 $\to$ SAM等との統合が今後の課題
}
