**# Object-Centric World Models Project

## 📌 Project Overview (プロジェクトリストの情報)
本プロジェクトでは、物体中心表現（Object-Centric Representation）を用いた世界モデルの構築と、その実世界データへの適用を目指します。
従来のSlot Attentionが抱えるスケーラビリティの問題（複雑なテクスチャや背景での精度低下）を解消し、プランニングや下流タスクでの有効性を検証します。

### Key Objectives
1.  **Scaling Object-Centric Learning**: 実写画像や複雑なシーンに対応可能な表現学習手法（DINOSAUR等）の実装・検証。
2.  **World Modeling**: 獲得した表現を用いたダイナミクスモデル（Transition Model）の構築。
3.  **Planning & Control**: 世界モデルを用いたプランニング精度の評価。
